This code is from DS535 Social Network Analysis Project
"Community Detection: The Case Study of the Relevance of Large Facebook Page in November 2017"
from Krich Prachyawatin
data from SNAP.Stanford.edu

1. download 3 files
- Jupeter Notebook file (Facebook-Page_SNA)
https://github.com/crazyipod/sna-project/blob/main/Facebook-Page_SNA.ipynb
- 2 CSV files (fb_edges_edit.csv & fb_features_edit.csv)
https://github.com/crazyipod/sna-project/blob/main/fb_edges_edit.csv
https://github.com/crazyipod/sna-project/blob/main/fb_features_edit.csv
2. download Anaconda Installer
https://www.anaconda.com/products/individual
3. download Gephi
https://gephi.org/users/download/
4. Install all programs
5. open Juperter notebook files
6. run all code in notebook
7. finish run and save gehf file
8. open Gephi and run gehf file
9. explore data
